# cServer
A simple custom built Web Server written in 2018.
* A single-threaded, asynchronous, non-blocking web server which serves static html files on a user defined port.

## Compile
From terminal:
```g++ main.cpp -o build/app```

## Run
From terminal:
```.build/app```
